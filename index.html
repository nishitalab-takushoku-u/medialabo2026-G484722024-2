数当てゲーム

概要

この課題では、JavaScript のイベント処理と DOM 操作を利用して数当てゲームを作成した。

  • プログラム開始時に 1～10 の範囲で乱数を生成し、それを正解の数として設定する。
  • プレイヤーはテキストボックスに予想した数を入力し、回答ボタンを押して判定を行う。
  • 入力された値は整数に変換して正解と比較し、正解の場合は成功メッセージを表示する。
  • 不正解の場合は、答えが入力値より大きいか小さいかのヒントを表示する。また、予想回数をカウントして画面に表示し、3回以内に正解できなかった場合はゲーム終了として正解を表示する。3回以上の回答を行った場合、ゲームが終了していることを警告として表示する。
  • これにより、イベントハンドラの登録、入力値の取得、条件分岐、画面表示の更新などの基本的な JavaScript の機能を学習した。

第1回 WWW と HTML

  • 今回学習したのはHTMLについての基礎事項。HTMLとはWebページの構造を記述するための書式であり、これらの文書を作ることによってWebページに掲載するコンテンツと、ページの基本的なレイアウトを設定できる。
  • <p>タグや<strong>タグ、<ul>や<ol>などのリストタグの使い方やWebページにリンクや画像を埋め込む方法について学習した。

  • 演習の成果: 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

  • 今回の講義では、CSSの飾り付けについて基礎的なことを学習した。
  • CSSはWebページの見た目を装飾するための言語であり、HTMLで記述されたWebページの構造に対して、文字の色や大きさ、背景色、レイアウトなどを指定することができる。
  • color や font-family を使った基本的な修飾や、displayなどを利用したレイアウトの飾り付けについて学習した。

  • 演習の成果: 演習2-4:ウェブブラウザの紹介

第3回 JavaScript の基本

  • 今回の講義では、HTMLでのJavaScriptの活用についての基礎を学習した。
  • htmlファイルとは別にjsファイルを用意して、HTMLの<head>要素に<script>要素を追加することで、外部ファイルからHTMLでJavaScriptを使うことが可能になる。 また、エラーメッセージとデバッグツールからブラウザ上でエラーの確認ができる。
  • for-of 構文について
  • デバッグの方法について、デバッグツールを活用してブレークポイントで自動的にプログラムを止めさせることができることが分かった。

  • 演習の成果: 演習3-6:オブジェクトの参照

第4回 DOM と JavaScript
