各授業回の学習事項

第1回 WWW と HTML

  • 今回学習したのはHTMLについての基礎事項。HTMLとはWebページの構造を記述するための書式であり、これらの文書を作ることによってWebページに掲載するコンテンツと、ページの基本的なレイアウトを設定できる。
  • <p>タグや<strong>タグ、<ul>や<ol>などのリストタグの使い方やWebページにリンクや画像を埋め込む方法について学習した。

  • 演習の成果: 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

  • 今回の講義では、CSSの飾り付けについて基礎的なことを学習した。
  • CSSはWebページの見た目を装飾するための言語であり、HTMLで記述されたWebページの構造に対して、文字の色や大きさ、背景色、レイアウトなどを指定することができる。
  • color や font-family を使った基本的な修飾や、displayなどを利用したレイアウトの飾り付けについて学習した。

  • 演習の成果: 演習2-4:ウェブブラウザの紹介
